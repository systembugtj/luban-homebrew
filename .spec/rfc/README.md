# RFC（Request for Comments）

本目录存放项目技术决策与规范的 RFC 文档。

## 命名与编号

- 文件名：`NNN-short-name.md`，其中 `NNN` 为三位数字（如 001、002）。
- 新 RFC 使用下一个未占用编号。

## 当前 RFC 列表

| 编号 | 标题 | 状态 |
|------|------|------|
| [001](001-linglong-homebrew-tap.md) | Linglong Homebrew Tap | Accepted |

## 工作流

- 新建或修改 RFC 时，请更新本 README 的「当前 RFC 列表」。
- 实现或废弃某 RFC 时，在文档内更新「状态」并在列表中同步。
